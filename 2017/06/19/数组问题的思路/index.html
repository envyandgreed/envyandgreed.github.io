Envy

Java

看一到算法题：Given an array nums, write a function to move all 0‘s to the end of it while maintaining the relative order of the non-zero elements.

For example, given nums = [0, 1, 0, 3, 12], after calling your function, nums should be [1, 3, 12, 0, 0].

上述代码需要1ms的时间运行，思路是将非零数字存储到一个辅助数组中，然后最后将后面空缺的部分全部填写为零，这个算法的时间复杂度和空间复杂度为O(n)。
