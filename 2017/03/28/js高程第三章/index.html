第三章基本法

语法

1.变量函数名以及操作符，都区分大小写。
2.标识符必须以字母、下划线或$开始。
3.严格模式，’’use strict’’可以用于在函数声明内部或者文件顶部。

数据类型

1.有六种:null,undefined,boolean,number,string,object。
2.typeof可以返回undefined\boolean\string\number\object\function，也是6种，但是少了null因为归类在了object里，因为null指向了空对象，多了一个function。
3.typeof可以typeof message 或者 typeof (message)。

  • undefined: 未经声明的变量也是undefined

  • null

  • boolean:可以转换为true：任何非空字符、任何非零字符包括infinity、任何对象以及n/a。可以转换为false：空字符、0和Nan、null以及undefined。

  • number

  1. 0.1 + 0.2 == 0.3 //false
  2. NaN表示一个本来要返回数值的数未返回数值，任何涉及NaN的操作都会返回NaN。
  • string：转换为字符串有两种方式
    1.toString可以指定基数，默认是10
    2.String可以正常调用toString，此外对null和undefined也滋瓷

var s1 = ‘01’
var s2 = ‘z’
s1 = +s1 //1
s2 = +s2 //NaN

var ov = 2; //10
var nv = ov << 5; //1000000
var nv2 = nv >> 5; //10

!Infinity //false
!NaN //true
!’’ //true
//&& 如果第一操作数是对象，则返回第二个操作数，如果第二个操作数是对象，只有在第一个操作数求值结果true的情况下才会返回该对象，如果都是对象则返回第二个操作数，如果有一个是null返回null，是NaN返回NaN，是undefined返回undefined。
//|| 如果第一个操作数是对象，则返回第一个操作数，如果第一个求值结果为false则返回第二个，如果都是对象则返回第一个，如果都是null返回null，是NaN返回NaN，是undefined返回undefined。
